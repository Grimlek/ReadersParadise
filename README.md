### ReadersParadise
Book Store Web Application      

#### Technology Stack
- Mysql
- JSTL
- JSP
- Servlets
- EJB
- JPA + Eclipselink
- Glassfish
- JQuery
- JQuery Validation Plugin
- CSS
- HTML

- Post / Redirect / Get
- Factory 
- Strategy
- Front Controller
